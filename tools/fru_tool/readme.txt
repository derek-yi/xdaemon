## 命令功能
Usage:
  fru_get            -- show fru info
  fru_load           -- load json fru
  fru_set_mac        -- set fru mac addr
  fru_set_rf_cal     -- set RF calibration param
  fru_set_sku        -- set fru SKU ID
  fru_set_uuid       -- set fru UUID

## fru_load
json文件格式参考fru_dfb.json，fru_rfb.json

## 注意
系统启动过程会运行cpri-r21-mul-slave.sh脚本，该脚本会修改eeprom数据：
echo "${eep_tmp:0:35}${uuid_tmp};" > /sys/devices/soc0/amba/e0004000.i2c/i2c-0/i2c-3/3-0054/eeprom

建议使用git库的app软件包：
http://facc1:3000/NR_PUB/rootfs_fhk_rru
